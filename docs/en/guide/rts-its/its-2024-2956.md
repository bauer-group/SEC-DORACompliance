# ITS (EU) 2024/2956 – Information Register

| Property | Value |
|---|---|
| **Number** | Implementing Regulation (EU) 2024/2956 |
| **DORA Article** | Art. 28(9) |
| **Pillar** | P4 – Third-Party Risk |
| **Publication** | 29.11.2024 |
| **Entry into force** | 22.12.2024 |
| **EUR-Lex** | [Link](https://eur-lex.europa.eu/eli/reg_impl/2024/2956) |

## Content

Standard templates for the information register of ICT third-party service provider contracts:

| Template | Content |
|---|---|
| B_01 | General information on the financial entity |
| B_02 | Contractual arrangements |
| B_03 | ICT third-party service provider identification (LEI/EUID) |
| B_04 | ICT service information |
| B_05 | Critical/important functions |
| B_06 | Subcontracting information |
| B_07 | Data processing locations |

**Identifier:** LEI or EUID (per EU Commission decision)
**First submission to BaFin:** 30.04.2025
